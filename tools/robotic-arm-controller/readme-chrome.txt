Chrome (Most Common):

Look for the shield icon in the address bar (left side, near the URL)
If no shield icon appears, try loading the page first, then:

Click the lock icon next to the URL
Look for "Site settings" or "Permissions"
Find "Insecure content" and set to "Allow"


Alternative method:

Go to chrome://settings/content/insecureContent
Click "Add" next to "Allow"
Enter your GitHub Pages URL (e.g., https://yourusername.github.io)


Developer method:

Press F12 (Developer Tools)
In Console, you'll see the mixed content error
Click "Allow" in the error message if available
